# A Genealogy Recording Application

An App meant to record and safely store genealogy data

# How To Run

- clone the project

## To run the server

```

 cd server
 npm install
 npm run server
```

## To run the server

```

 cd client
 npm install
 npm run dev
```

## Notice !!!

when adding parents,the names should match Exactly those shown on the members table

## Technologies Used

- React js
- Tailwind Css
- Express Js
- VsCode IDE
- Nodejs
